Main.py is a python file that reads in covid_DNA.fasta and processes the DNA sequence to match with a given DNA sequence. Uses Dynamic programming to 
calculate if the sequence matches

DNAMatching.ipynb is a google COLAB file. This algorithm was originally developed inside of Google COLAB's online IDE. DNAMatching.ipynb is the original
file that wasnt modified to work outside of Google COLAB.

main.py and covid_DNA.fasta need to be in the same folder in order for script to work.
